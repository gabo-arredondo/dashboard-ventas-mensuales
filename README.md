
 📊 Dashboard de Ventas Mensuales

## 📌 Descripción del proyecto
Este proyecto consiste en un dashboard web que permite visualizar y analizar las ventas mensuales de un negocio de electrodomésticos.

El objetivo principal es mostrar de forma clara el comportamiento de las ventas a lo largo del tiempo, así como identificar qué categorías de productos generan mayores ingresos.

---

## 🎯 Pregunta de negocio
> ¿Cómo se comportaron las ventas del negocio a lo largo de los meses y qué categorías generaron más ingresos?

---

## 📁 Estructura de datos
Cada registro representa el total de ventas de una categoría en un mes específico.

Los datos utilizados contienen los siguientes campos:

- **mes**: Mes de la venta (Ej: Enero, Febrero, etc.)
- **categoria**: Tipo de producto (Cocinas, Lavadoras, Refrigeradores)
- **cantidad**: Número de unidades vendidas
- **total**: Ingreso total generado (cantidad × precio)

Los datos están agregados por mes y categoría.

---

## 📊 Métricas del dashboard

El dashboard utiliza las siguientes métricas principales:

- ✅ Total de ventas (ingresos)
- ✅ Total de unidades vendidas
- ✅ Ventas por mes
- ✅ Ventas por categoría

---

## 🔎 Funcionalidades

- Visualización de ventas mensuales mediante gráfico de barras
- Resumen general con ingresos totales y unidades vendidas
- Tabla detallada con información por mes y categoría
- Filtro por mes para analizar datos específicos

---

## 📈 Principales insights

A partir del análisis de los datos se puede observar que:

- Existen variaciones en las ventas a lo largo del año
- Algunos meses presentan mayor rendimiento que otros
- Se pueden identificar temporadas altas y bajas
- Las categorías de productos contribuyen de forma diferente a los ingresos
